# 🛡️ SafeVista

SafeVista is a responsive e-commerce MVP inspired by industrial safety stores such as BuySafetyPosters. The project demonstrates frontend development skills using pure HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive Homepage
* Product Categories
* Featured Products Section
* Best Sellers Section
* Product Detail Page
* Shopping Cart
* Checkout Interface
* LocalStorage-based Demo Authentication
* Mobile-Friendly Design
* Clean Industrial UI

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* LocalStorage

## 💻 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/safevista.git
```

### Run Locally

Simply open `index.html` in your browser.

No additional dependencies or installations are required.

## 📱 Responsiveness

The project is optimized for desktop, tablet, and modern mobile devices. Further improvements can be made to enhance compatibility with smaller screen sizes.

## 🔒 Authentication

This project includes a demonstration authentication system implemented using LocalStorage for educational and prototype purposes. It is not intended for production use.

## 🎯 Purpose

This project was developed as a frontend MVP to showcase web development fundamentals, UI design, and client-side functionality using vanilla technologies.
